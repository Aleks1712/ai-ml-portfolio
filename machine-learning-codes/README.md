# Machine Learning Codes

Samling av kode + notebooks (Python og R) organisert etter kursdeler (preprocessing, regression, classification, clustering, NLP, deep learning, osv.).

## Struktur

- Hver `Part .../Section .../Python/` inneholder `.ipynb` + `.py` og ofte en `.csv` dataset.
- Tilsvarende finnes `R/`-mapper med `.R`-scripts og datasett.

## Kjøre lokalt

- Åpne en notebook i `Python/`-mappene og kjør cellene.
- Mange eksempler bruker standardpakker som `numpy`, `pandas`, `matplotlib`, `scikit-learn` (og noen ganger `tensorflow`/`xgboost`).

## GitHub tips

- Repoet inkluderer `.gitignore` som ignorerer `.DS_Store`, notebook checkpoints, og `.Rhistory`.
